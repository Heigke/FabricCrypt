Could an attacker fake this? We tried three attacks. First: replay. Record a real ikaros signature, then replay it later. Rejected ninety-nine point four percent of the time. Each verification uses a fresh challenge nonce, so the recorded signature is for the wrong challenge. Second: a random signature. Rejected ninety-nine point four percent. Random data does not match any real chip. Third: brute force. The attacker would need to record a valid signature for every possible challenge, which is two to the sixty-third, roughly nine quintillion. Infeasible.
